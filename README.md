# Student Grade Tracker

## Overview

Student Grade Tracker is a Java console-based application that helps manage student records and track their academic performance.

The project allows users to:

- Add student details
- Display all students
- Search students by name
- Calculate grades automatically
- Store student records using ArrayList
- Practice Object-Oriented Programming concepts in Java

---

## Features

### Add Student
Users can enter:
- Student Name
- Student Marks

### Display Students
Shows:
- Name
- Marks
- Grade

### Search Student
Search a student by name and display their details.

### Grade Calculation

| Marks | Grade |
|---------|---------|
| 90 - 100 | A |
| 80 - 89 | B |
| 70 - 79 | C |
| 60 - 69 | D |
| Below 60 | F |
